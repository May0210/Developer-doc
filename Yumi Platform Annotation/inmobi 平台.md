###  [Inmobi](https://www.inmobi.com) 平台
  

1. APP ID

   在[Inmobi 活动](https://www.inmobi.com/portal/#properties) 页面，选择您的应用，获取APP ID参数。

2. 位置ID

   在[Inmobi 属性](https://www.inmobi.com/portal/#properties)页面，选择您的应用，在位置下获取位置ID。

3. 注册邮箱  
登录[Inmobi 平台](https://www.inmobi.com)使用的账户信息。

4. 	密码    
登录[Inmobi](https://www.inmobi.com)账户所需要的密码，inmobi需要登录密码才可访问其API。

5. API Key

    在[InMobi 账户设置 -API Key](https://www.inmobi.com/portal/#user/apiKey)页面，通过电子邮件申请一个API密钥。

6. 结算货币

   在[Inmobi 收入](https://www.inmobi.com/portal/#earnings)页面，获取结算币种。此币种即注册时选择的账号币种，用于汇率转换。
